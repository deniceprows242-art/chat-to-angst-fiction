# Evidence Rules for Chat-Based Angst Fiction

## Core Principle

Every sentence must survive the question: **"Where in the chat record does this come from?"**

If the answer is "nowhere", it does not go in.

## What Counts As Evidence

### Tier 1: Direct Quote

Verbatim text from a message. It can be used directly in dialogue or narration.

Example: if the record says `"以后别等我了"`, the story may write: 她说："以后别等我了。"

### Tier 2: Observable Pattern

Changes visible across multiple messages: reply gaps, message length, word choices, frequency, initiative, repeated avoidance, or changes in address.

Example: 她的回复在变短. Cite the earlier long replies and the later one-word replies.

### Tier 3: Statistical Fact

Numbers computed from the record: message totals, sender ratios, time spans, longest silence, first/last message, daily frequency, or repeated keywords.

Example: 三年里一共有 12,483 条消息，其中一方发了 7,102 条.

### Tier 4: Reasonable Emotional Inference

What the narrator likely felt, based on their own words and behavior patterns. It must be signaled with "大概", "也许", "可能", "像是", or similarly cautious language.

Example: "他盯着那个句号，大概看了很久" requires supporting evidence such as a long gap before the next reply plus a clear tone shift.

### Tier 5: Real Artifact

Objects, links, images, stickers, voice notes, nicknames, gifts, games, screenshots, or recalled messages explicitly mentioned in the chat.

Example: if a music box appears in the chat, the story may use it as an ending artifact.

## What Is Not Evidence

| Category | Example of violation | Why forbidden |
|---|---|---|
| Physical description | "她穿着一件浅色羽绒服" | Not in the record |
| Sensory detail | "他闻到洗衣液的味道" | Not in the record |
| Other person's internal state | "她觉得愧疚" | She did not say this |
| Setting description | "窗帘没拉严，走廊的灯漏进来" | Not in the record |
| Future events | "他会遇到一个新的人" | Pure speculation |
| Body language | "她笑得眼睛弯弯的" | Only allowed if someone described it in the source |
| Weather | "那天下着雨" | Only allowed if the messages mention rain |

## Boundary

The safest inner world to write is the narrator's, and even that should come from their messages, repeated behavior, and silence. The other person's inner world is a black box unless they explicitly wrote it.

Uncertainty is often the emotional truth. "他不知道她有没有喜欢过他" can be stronger than inventing an answer.

## Evidence Map Template

Use a private evidence map while drafting. Do not publish maps that contain real chat content.

| Narrative claim | Evidence type | Source |
|---|---|---|
| Opening timestamp | Direct quote / timestamp | `2024-10-19 08:45:12` |
| Reply became shorter | Observable pattern | Compare messages A, B, C |
| Narrator waited before replying | Observable pattern | Gap between message X and Y |
| Ending object | Real artifact | Message mentioning the object |
